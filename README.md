This was build with

<ul>

    <li>SCSS/SASS</li>
    <li>Typescript</li>
    <li>Webpack</li>
    <li>Babel</li>

### Install dependencies

```
npm install
```

### Run in development

```
npm run dev
```

### Run in production

```
npm run build

then open index.html in the dist folder
```
